Task

Write tested code for a REST endpoint (use any framework you like) that accepts a numeric type and returns a truncated, "prettified" string version. The prettified version should include one number after the decimal when the truncated number is not an integer. It should prettify numbers greater than 6 digits and support millions, billions and trillions. You should use short scale numbers to output the prettified number.
Examples:
● input: 500 output 500
● input: 3400 outputs 3.4K
● input: 1000000 output: 1M
● input: 2500000.34 output: 2.5M
● input: 1123456789 output: 1.1B
